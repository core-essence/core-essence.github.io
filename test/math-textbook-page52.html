P.52

不等式と文字式の表現

問題 35 (3)
200ページの本を読む。昨日 x ページ読んで、今日 y ページ読んだ。まだ読み終えていない残りのページ数を式で表しなさい。
「なぜ 200 - (x + y) になるのか」を、できるだけ具体的にイメージで説明します
①本の全体
本は 200ページ。これが「スタート時点での総量」です。
②昨日読んだページ数
昨日読んだ量は x ページ。
だから、200ページのうち x ページ分はもう消費された（残りから減った）ことになります。
③今日読んだページ数
今日読んだ量は y ページ。
だから、さらに y ページ分が消費されたことになります。
④読んだページの合計
昨日と今日で読んだページを合わせると x + y ページ読んだことになります。
⑤残りページ数
全体200ページから、すでに読んだ分 (x + y) を引けば、残りが出ます。
200 - (x + y)
これが「まだ読んでいないページ数」です。
日常の例でたとえると
2000円持っていて、昨日 x 円使った。今日 y 円使った。残りの所持金は？
と聞かれたら、自然に 2000 - (x + y) になりますよね。
本のページも「総量から使った（読んだ）分を引く」という考え方で同じなんです。
✅ まとめ
「200 - (x + y)」は
• 200ページが全体
• xページとyページをもう読んだ
• だから全体から読んだ分を引く
という流れで出てきた式なんです。
問題 35 (4)
b は a の4倍より1小さい。
日本語をほどく
• a の4倍 …… a を4回分 ⇒ 4a
• より1小さい …… そこから1だけ減らす ⇒ 4a - 1
• bは〜である …… b がその値と等しい ⇒ b = ...
できた式：b = 4a - 1
たしかめ例
a = 5 なら
4a = 20、1小さい → 19。だから b = 19。
式に入れても b = 4×5 - 1 = 19。合ってるね。
問題 35 (5)
x から y をひいた差は 5 より大きい。
日本語をほどく
• x から y をひいた差 …… 引き算の結果 ⇒ x - y
• 5 より大きい …… > 5
できた式：x - y > 5
たしかめ例
x = 12, y = 6 → x - y = 6。6は5より大きい → 成り立つ
x = 10, y = 6 → x - y = 4。4は5より大きくない → 成り立たない
問題 3
縦が a cm、横が b cm の長方形がある。このとき、不等式 a + b < 60 はどんなことを表していますか。
まず意味を整理
• a … 縦の長さ（cm）
• b … 横の長さ（cm）
• a + b … 縦と横を足した長さ（「一辺ずつの合計」）
※ 周の長さ（ぐるっと一周）は 2a + 2b。ここは周でも面積でもないよ。
あくまで「縦＋横」という和に注目した不等式。
不等式の読み
a + b < 60 …… 「縦と横の合計が 60cm より小さい」
文章にすると：
縦の長さと横の長さの和が 60cm より小さい
たしかめ例
a = 20, b = 30 → a + b = 50 < 60 → OK
a = 35, b = 28 → a + b = 63 < 60 ではない → NG

〜より小さい <

〜以下 ≤

〜から…をひいた差 −
