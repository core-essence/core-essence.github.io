①重さの内訳を整理
• 原稿用紙1枚：a g
• 原稿用紙16枚：16a g
• 封筒：b g
②全体の重さは？
原稿用紙16枚 + 封筒 = 16a + b g
③「250g以上」を式で表す
全体の重さ ≥ 250
つまり：16a + b ≥ 250
答え：16a + b ≥ 250

(1)基準日から n 週間後の週の金曜日は、基準日から何日後になるか、n を使った式で表しなさい。
週と日の関係を理解しよう
①1週間 = 7日を使って考える
• n週間 = 7n 日
②基準日（3月1日）は月曜日
• 同じ週の金曜日：4日後（3月5日）
• 1週間後の金曜日：4 + 7 = 11日後（3月12日）
• 2週間後の金曜日：4 + 14 = 18日後（3月19日）
③パターンを見つける
n週間後の金曜日 = 4 + 7n 日後
= 7n + 4 日後
別の考え方：
基準日から金曜日まで：4日
そこからn週間分進む：+7n日
合計：4 + 7n = 7n + 4 日後
答え：7n + 4 日後
検証してみよう
• n = 1のとき：7×1 + 4 = 11日後（3月12日） ✓
• n = 2のとき：7×2 + 4 = 18日後（3月19日） ✓
(2)彩さんの誕生日は、基準日から204日後の9月21日である。この日は、何週間後の週の何曜日であるか求めなさい。

①204日は何週間と何日？
204 ÷ 7 = 29 あまり 1
つまり：29週間と1日
②基準日（月曜日）から1日後は？
月曜日の1日後 = 火曜日
